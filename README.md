# interval-refactor-project
Refactor windows forms single form c# app to MVC style using observer (not event).
A .NET 5 c# winforms desktop app

Does not change the way the app starts, or the way it references other forms.
It is meant to be modified further to suit your purpose.
Please consider this code as a starting place for you own project 

In his book Martin Fowler (2014) suggests that after this refactoring the Gui can easily be replace with another, which is what I intend to use this for.
His examples are in Java and AWT suggesting that AWT can be replaced by Swing when the refactoring is complete.

Reference
Fowler, Martin et al. (2014) Refactoring Improving the design of existing code.  Adison-Wesley. 29th printing, Chapter 8, Duplicate Observered Data.
